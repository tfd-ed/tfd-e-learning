# TFD E Learning

## Setup Guide
Clone the repository
```bash
git clone --recurse-submodules https://github.com/tfd-ed/tfd-e-learning.git
```
Make sure you have Docker installed.
```bash
$ docker-compose build
$ docker-compose up -d
```
Go to
```bash
http://localhost
```
